# Launch copy

Ready-to-use posts, one file per platform under [`platform/`](platform/). Swap
in real links, tweak tone per community, and always disclose you're the dev.

All copy leads with the same story: scattered tools, work that was never really
saved, and notes that bled away across device changes — answered by "everything
is a real file in a git repo you own."

## One-liner / tagline

> A self-hosted, single-binary workspace for code, drawings, mindmaps, docs and
> kanban boards — every note is a real file the instant you type it, synced to
> your own git repo, with expiring share links.
